# LP
Lógica e Programação

Repositório criado para registrar minha evolução no estudo de lógica de programação, algoritmos e linguagem C.

Os exercícios foram desenvolvidos durante a graduação e abrangem tópicos como:

• Estruturas de decisão
• Estruturas de repetição
• Vetores
• Matrizes
• Funções
• Ordenação
• Estruturas de dados

O objetivo deste repositório é documentar o processo de aprendizagem e a evolução das soluções ao longo do tempo.
